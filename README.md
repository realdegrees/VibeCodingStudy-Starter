**Grundlegendes:**
- Verwende bitte ausschließlich html, css und javascript
- Benutze für die Entwicklung den Github Copilot

**Folgende Anforderungen müssen umgesetzt werden:**
- Suchfeld für Stadtname
- Aktuelles Wetter: Temperatur, Icon, Beschreibung, Stadtname
- Zusätzliche Infos: Luftfeuchtigkeit, Wind, gefühlte Temperatur
- 5-Tage-Forecast als visuelle Darstellung
- Filter für Forecast
- Dynamisches Styling passend zum Wetter
- Stündliche Vorhersage
- Sauberes Layout mit klarer Struktur
- Grundlegendes Styling
- Einheitenwechsel (°C/°F)

**Um die Anforderungen umsetzen zu können, benutze bitte diese API:** [Link zur API](https://open-meteo.com/en/docs)
